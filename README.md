# StayEase – Smart PG Management System

StayEase is a cross-platform mobile application built using Flutter and Firebase that simplifies and automates the management of Paying Guest (PG) accommodations. It provides a digital solution for efficient communication, secure operations, and real-time data management between residents and PG owners.

---
## Overview

Managing PG accommodations manually leads to issues like poor record-keeping, payment confusion, communication gaps, and security risks.

StayEase solves these problems by providing a centralized digital platform that ensures:
- Transparency in operations  
- Improved security  
- Efficient management  
- Better user experience  

---

## Features

### Resident Features
- User Registration & Login  
- Search PGs by location  
- View PG details with map integration  
- QR Code-based check-in/check-out  
- Meal attendance tracking  
- Rent payment & history  
- Raise and track complaints  
- Rate and review PGs  

---

### Admin / PG Owner Features
- Manage PG listings and rooms  
- Add and manage residents  
- Track payments and records  
- Update meal menus  
- Complaint management system  
- Maintain visitor logs  
- Send notifications and updates  

---
